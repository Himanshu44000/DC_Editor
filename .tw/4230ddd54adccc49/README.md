# React + TypeScript + Vite + Tailwind CSS

Production-ready frontend starter with modern defaults and VS Code-friendly tooling.

## Environment Provided

- React 19 + React DOM 19
- TypeScript 5 (strict mode)
- Vite 7 + `@vitejs/plugin-react`
- Tailwind CSS 4 via `@tailwindcss/vite`
- ESLint 9 with TypeScript + React hooks rules

## Quick Start

```bash
npm install
npm run dev
```

## Common Commands

```bash
npm run typecheck
npm run lint
npm run build
npm run preview
```

## Notes

- Tailwind v4 is enabled through the Vite plugin and `@import "tailwindcss";` in `src/index.css`.
- This template is intentionally minimal so you can start building immediately without removing demo complexity.
